+ 阿里的Java
  + 所有阿里及其附属公司开发的Java应用都是运行在AJDK
    + 面向服务的架构
      + 服务之间的通信使用RPC
      + 通常，每个服务有多个JVMs组成的集群
    + 异构
      + C／C++写的本地库(the native libraries)和Java之间使用JNI通信
  + 在海量规模上运行Java
    + <span style="border-bottom:2px solid red;">百万的JVM实例，每秒可以响应大量的请求</span>
